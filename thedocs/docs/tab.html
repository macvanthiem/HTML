Tab

A tab keeps related content in a single container that is shown and hidden through navigation.

Horizontal

Basically, you can create a tab using any type of nav by adding few data attributes and companying it with a .tab-content.

Home Profile Contact
Tab 1 - Home
Tab 2 - Profile
Tab 3 - Contact
<nav class="nav nav-tabs" id="tab1" role="tablist">
  <a class="nav-item nav-link active" id="nav-home-tab" data-toggle="tab" href="#tab1-home" role="tab" aria-controls="nav-home" aria-selected="true">Home</a>
  <a class="nav-item nav-link" id="nav-profile-tab" data-toggle="tab" href="#tab1-profile" role="tab" aria-controls="nav-profile" aria-selected="false">Profile</a>
  <a class="nav-item nav-link" id="nav-contact-tab" data-toggle="tab" href="#tab1-contact" role="tab" aria-controls="nav-contact" aria-selected="false">Contact</a>
</nav>

<div class="tab-content p-4">
  <div class="tab-pane fade show active" id="tab1-home" role="tabpanel" aria-labelledby="home-tab">Tab 1 - Home</div>
  <div class="tab-pane fade" id="tab1-profile" role="tabpanel" aria-labelledby="profile-tab">Tab 2 - Profile</div>
  <div class="tab-pane fade" id="tab1-contact" role="tabpanel" aria-labelledby="contact-tab">Tab 3 - Contact</div>
</div>

Vertical

You can use Bootstrap grids to create vertical tabs.
